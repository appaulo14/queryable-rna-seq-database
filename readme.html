Purpose

Formally known as the Queryable RNA-Seq Database, this system is designed to simplify the process of RNA-seq analysis by providing the ability upload the result data from RNA-Seq analysis into a database, store it, and query it in many different ways.

Overview

The purpose of the system is to automate and simplify as much as possible the process of analyzing RNA-Seq results data by storing it in a database and providing many options for querying it.

Goals

  • Goal 1: Provide a system through which Biologists can analyze their RNA-Seq results data, specifically differential expression tests, novel transcript discoveries, and assembled transcripts.
  • Goal 2: The system should allow the user to get meaningful results with minimal learning time. For this goal to be satisfied, two senior Biologists familiar with RNA-seq must approve the system.

Features

Helpful Tips Integrated With the User Manual

To reduce the learning curve, helpful tips and links to relevant sections of the user manual and other external websites with important information such as the Cufflinks manual are distributed throughout the Queryable RNA-Seq Database.

GO Terms Are Automatically Found For Uploaded Data

When you upload your data, the Queryable RNA-Seq Database automatically runs blast2go to find all of the Gene Ontology (GO) terms for your transcripts. They can then be view on the query differentially expressed transcripts, querying differentially expressed genes, or query transcript isoforms page.

Upload from Multiple Differential Programs

Cuffdiff

If you have data produced by Cuffdiff, the differential expression tests, novel transcript classifications, and transcript fasta sequences can be uploaded for querying and blasting.

Trinity with EdgeR

If you have differential expression tests from Trinity with EdgeR, you can upload your transcript fasta sequences, transcript differential expression tests, and even gene differential expression tests if available.

"Generic Nucleotide Fasta Sequences

If you just want a Nucleotide Blast database to use to compare sequences, you can upload a nucleotide fasta sequence file from any program and a Blast database will automatically be created for it.

Query Your Data

Query Features

Sort The Results Table

By clicking on the green arrows in the header of each table column, the table can be sorted in either ascending or descending order.

A Table Sorted in Ascending Order By Transcript
A Table Sorted in Descending Order By Transcript
Download Your Results as A Text File

Click the "Save to Disk" button to save your results table to a text file. The sorting order will be preserved. Flash is required for this feature to work.

View The Fasta Sequence For A Transcript

To view the fasta sequence for a transcript, simply click on it.

View Transcript Fasta Sequence Associated With a Gene

To view the transcript fasta sequences for the transcripts associated with a gene, click on that gene.

View Gene Ontology (GO) Term Information

Clicking on a GO Term will take you to the Gene Ontology web site's entry for that GO term.

Query Your Transcript Differential Expression Tests

View differential expression tests for your transcripts. You can select the dataset and the two samples you want to see the differential expression tests for. You can also filter the query in various to only see certain results. The query can be filtered by Gene Ontology (GO) Terms, GO IDs, and transcript name. The results are displayed in a sortable and downloadable table.

Query Your Gene Differential Expression Tests

View differential expression tests for your genes. You can select the dataset and the two samples you want to see the differential expression tests for. You can also filter the query in various to only see certain results. The query can be filtered by Gene Ontology (GO) Terms, GO IDs, and transcript name. The results are displayed in a sortable and downloadable table.

Query Your Transcript Isoforms (Cuffdiff Only)

View if you dataset has any novel transcript isoforms or other interesting transcript classifications. This page also lets you view the FPKM information for the selected sample. You can also filter the query in various to only see certain results. The query can be filtered by Gene Ontology (GO) Terms, GO IDs, and transcript name. The results are displayed in a sortable and downloadable table.

Blast a Blast Database of Your Sequences

The Queryable RNA-Seq Database allows you to query your data using the Megablast program of Blastn, using TBlastn, or using TBlastx. These blast query pages have a large subset of the features and algorithm settings available on the Blastn page, TBlastn page, and TBlastx page respectively of the NCBI Blast website.
